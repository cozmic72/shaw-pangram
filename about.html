Shavian Pangram & Anagram Solver

A web-based tool for finding anagrams or perfect pangrams (using each letter exactly once) for the Shavian alphabet.

Features

  • Pangram Solver: Find perfect pangrams for the full 48-letter Shavian alphabet or custom letter sets
  • Anagram Solver: Solve arbitrary anagrams starting from arbitrary words or phrases
  • Interactive UI: Drag-and-drop result reordering and auto-pause
  • Save Solutions: Store your favorite (partial) solutions for later use

How to Use

  1. Target Letters: Leave empty for all 48 Shavian letters, or specify custom words or phrases
  2. Solve Words: Words to include in your solution - use this to narrow down the solution space
  3. Exclude Words: Words to exclude from results
  4. Drag & Drop: Reorder results by dragging words to build sentences

Algorithm

The solver uses a Trie-based backtracking algorithm optimized for pangram solving:

  • Trie construction of all valid words using only target letters
  • Letter rarity ordering to prune the search space early
  • Recursive backtracking to explore word combinations
  • Anagram grouping to avoid duplicate searches

Dictionary

This project uses the readlex dictionary, which contains English words transcribed in the Shavian alphabet.
